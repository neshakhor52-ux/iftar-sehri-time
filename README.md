# Offline Sehri & Iftar Telegram Bot (BD 2026)

এই বট **ইন্টারনেট API ছাড়াই** কাজ করবে, কারণ সব জেলার সময় `iftar_sehri_2026_bd_districtwise.json` ফাইলে রাখা আছে।

## Run
```bash
pip install -r requirements.txt
export BOT_TOKEN="YOUR_TOKEN"
python bot.py
```

## Use
- `/today ঢাকা`
- `/date 2026-03-05 ঢাকা`
- অথবা শুধু লিখুন: `ঢাকা`

ডাটা: ইসলামিক ফাউন্ডেশন, জেলা-ভিত্তিক রমজান ২০২৬ সময়সূচি।
